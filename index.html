SVG

É a abreviatura de Scalable Vector Graphics que pode ser traduzido do inglês como gráficos vetoriais escaláveis. Trata-se de uma linguagem XML para descrever de forma vetorial desenhos e gráficos bidimensionais, quer de forma estática, quer dinâmica ou animada. Umas das principais características dos gráficos vetoriais, é que não perdem qualidade ao serem ampliados. A grande diferença entre o SVG e outros formatos vetoriais, é o fato de ser um formato aberto, não sendo propriedade de nenhuma empresa. Foi criado pela World Wide Web Consortium, responsável pela definição de outros padrões, como o HTML e o XHTML.

HTML

é uma linguagem usada para desenvolver websites que significa Linguagen de Marcação de Hypertexto.

CSS

É uma linguagem utilizada para definir a aparência dos documentos que utilizam linguagem de marcação

JAVASCRIPT

É uma linguagem de programação utilizada para controlar o HTML e o CSS para manipular o comportamento na página

JSON

É um formato de troca de dados entre sistemas independente de linguagem de programação derivado do JavaScript
